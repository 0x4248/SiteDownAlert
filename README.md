# SiteDownAlert

A simple webpage that takes in parameters and shows a nice warning message to the user.

## Usage

```html
DOMAIN.COM/?message=MESSAGE_IN_BASE64&link=LINK_IN_BASE64&link_text=LINK_TEXT_IN_BASE64
```
